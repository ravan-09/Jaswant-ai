Concept:

  • The neither-nor case is applicable when both the conclusions are false, whether definite or not.
  • The either-or case is applicable for a pair of statements only if either of the following pair of keywords is used AND only if the following conditions are satisfied:

Keywords:

  1. All and some not
  2. Some and no

Conditions:

  1. The same elements are used in the two conclusions. For example, in this case, the elements jungles and malls are common to both conclusions.
  2. Both conclusions should be possibilities. That is, they should be false because the conclusion is possible but not definite.

Then, the pair of conclusions are considered as complementary pairs.

The least possible diagram for the given statements is as follows

Conclusions:

I. Some chairs are bed. → False (it is possible but not definite)

II. No bed is a chair. → False (it is possible but not definite)

These are complementary conclusions

Hence, either I or II follows.
